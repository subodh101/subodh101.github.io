## Installation

Navigate to your themes folder in your Hugo site and use the following commands:

**Note : You must use `hugo-extended` version to compile SCSS**

```
  $ mkdir themes
  $ cd themes
  $ git clone https://github.com/themefisher/liva-hugo.git
```
